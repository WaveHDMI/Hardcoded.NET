# Hardcoded.NET

> **Turn "hardcoding" into a performance feature.**
>
> Ironic, but accurate. **Hardcoded.NET** is a Source Generator that leverages Roslyn to convert resources into mapped constants. It gives you the maintainability of external files with the raw performance of string literals.
